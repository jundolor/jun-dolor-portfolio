Mindoro Peaks/ Emojis API

The Emojis API at emoji-api-5802.onrender.com/api was developed in Node JS while the Mindoro Peaks API at mindoro-peaks.onrender.com/api was developed in Express JS.

The Emojis API provides a resource information for Emojis. The API returns the Emoji keyboard characters and descriptions.

The Mindoro Peaks API provides resource information for mountains in Mindoro. The API returns the following: latitude, longitude, elevation, prominence and description.
