Prompt Engineering for Developers

Conducted with the Mentor Group, the online workshop introduced developers to the emerging field of Prompt Engineering within DevOps.

Participants gained skills in effectively using AI prompts to create codes, know the limitations of AI, and generate test cases.

The session covered the basics of prompt engineering, presented practical use cases, and provided hands-on exercises. This ensured that the participants gained actionable skills they can immediately implement.
